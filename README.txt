
HOW TO UPLOAD TO GITHUB

1. Unzip this folder.
2. Log in to your GitHub account.
3. Open your existing repository.
4. Click "Add file" â†’ "Upload files".
5. Upload index.html and styles.css.
6. Click "Commit changes".

ENABLE GITHUB PAGES
1. In the repo, go to Settings.
2. Click "Pages".
3. Under Source, select "Deploy from branch".
4. Branch: main, Folder: /root.
5. Save.

Your site will be live at:
https://YOUR-USERNAME.github.io/REPO-NAME/

To replace images:
Swap the gray placeholders with <img src="yourimage.jpg"> tags.
